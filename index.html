CHAPTER 01

My Projects

Need to Go: Again!

ROLE — Lead Writer / Lead Programmer

Need to Go: Again! is a lighthearted, comedic branching narrative game that explores a broke college student desperately searching for a public restroom. Built using Unity + Yarn Spinner, this project was a design exercise in narrative choices and branching paths.

The hardest part of this project wasn't writing the jokes, it was making sure every choice felt like it mattered. Early drafts let players "win" no matter what they picked, which flattened the whole point of a branching narrative. I rebuilt the dialogue graph around three real failure states and mapped every node by hand in Yarn Spinner before writing a single line of final dialogue, which made it much easier to spot dead-end branches and dialogue that contradicted earlier choices.

Gun With It!

ROLE — Systems Design

Gun With It! is a fast-paced, simultaneous combat card game set in a gritty, post-apocalyptic wasteland controlled by the tyrannical El Feo. Players must strategize between using item cards or weapon cards in order to be the last one standing.

As this is a tabletop game, I learned a lot in the process of making this game. Index cards work great for fast iteration and prototyping, but aren't exactly pretty to look at. I had to learn the physical card-crafting workflow for this and am very proud of the end result (Though I had to print several re-tries)

The index-card prototypes went through many many rounds before a single card got real art. The "simultaneous reveal" mechanic was the riskiest part of the design, yet it turned out to be the most engaging. During playtests, players often counted down together, upping the excitement of the round. However, because all cards are revealed at the same time, this meant players spent more time parsing what just happened. We ended up color-coding the card backs by category (item vs. weapon) so players could read the game-state at a glance, which is a trick I will now reach for in future tabletop prototypes.
